Telugu Transliteration

Writing Telugu words using PramukhIME is as easy as writing our names in English. For quick help, checkout the following image. This image shows which letter(s) will be typed when you press a particular key (combination). Press F9 (or your preferred Shortcut Key) to switch between currently selected language and last selected language.
Note: Typing is case-sensitive. Typing small "a" and capital "A" will yield different letters. Make sure "Caps Lock" key is not ON.

Symbol

The English symbols [ ] { } ( ) < > - + / = ; . , ' " ? ! % \ _ & $ # ~ * ^ are transliterated into the same symbols.
Symbols : | @ have special meaning. You can type this way
: = :-

| = |-

@ = @-

These symbols will type Telugu characters first but if "-" will be followed, it will remove previously typed Telugu character and then type the symbol.

Rule

  1. Consonant is followed by vowel sign character to make a live consonant.
    Example
    క కౕ కా కి కీ కు కూ కృ కౄ కౢ కౣ కే కె కై కౖ కో కొ కౌ కం కఁ కః
    ka ke_ kaa ki kee ku koo kRi kRI kLi- kLI- kE ke kai kaI kO ko kau kaM kaM- ka:
  2. Two or more consecutive consonants make consonant cluster. The last consonant takes the full form and all preceding consonants within cluster take half form
    Example
    నమస్కారం = namaskaaraM

  3. Special consonant cluster forms (ligature) will be automatically created if it is available in the font. Look at the examples for clarification. All possible ligatures may not be given in the example. Try typing the consonant combinations according to ligature pronounciation and it will work. For example, try typing "vidhyaa", "viDyaa", "vidyaa" to get the correct ligature for "dy" in "vidyaa".
  4. Character "-" is followed by consonant will be transliterated as ్ (Halant/Viram)
    Example
    ఖ క్హ ఘ గ్హ ఛ చ్హ ఝ జ్హ ఠ ట్హ ఢ డ్హ థ త్హ ధ ద్హ ఫ ప్హ భ బ్హ శ స్హ ష శ్హ
    kha k-ha gha g-ha chha ch-ha jha j-ha Tha T-ha Dha D-ha tha t-ha dha d-ha pha p-ha bha b-ha sha s-ha Sha S-ha
  5. If two english characters are making one Telugu Vowel (i.e. ai, au), then "-" character is used to separate them into two different vowels.
    Example
    ఐరావత్ = airaavat

    అఇరావత్ = a-iraavat

  6. Character "m" transliterates to "ం" if it is the last character in the word.
    Example
    వర్షం = varSham

  7. Character "n"/"m" will be automatically transliterated to anusvar ం
    If character Followed by Transliterated to
    n k, kh, g, gh ం
    n ch, chh, j, jh ం
    n T, Th, D, Dh ం
    n t, th, d, dh ం
    m p, ph, b, bh, v ం
    n l, sh, Sh, s ం
    n - (Dash) న్
    m - (Dash) మ్
    n Any other character న్
    m Any other character మ్
    Example
    బంగారం = bangaaram

    మంజూరు = manjooru

    పండ్లు = panDlu

    చందన్ = chandan

    ప్రారంభ = praarambha

    సంస్కరణ = sanskaraNa

    సన్స్కరణ = san-skaraNa

ZWJ/ZWNJ character

ZWJ and ZWNJ characters are used to produce alternate rendering of ligature.
  1. A consonant followed by ZWJ character will produce half-formed consonant character.
    Example
    ప్రశ్న = prashna

    ప్రశ్‍న = prash--na

  2. A consonant followed by ZWNJ character will produce dead consonant character.
    Example
    ప్రశ్న = prashna

    ప్రశ్‌న = prash---na

Example

నమస్కారం = namaskaaraM

పక్షుల్ని = pakShulni

శ్రీ = shree

ఎక్స్ప్రెస = ekspresa

ఎక్స్‌ప్రెస = eks---presa
