Santali Transliteration

Writing Santali words using PramukhIME is as easy as writing our names in English. For quick help, checkout the following image. This image shows which letter(s) will be typed when you press a particular key (combination). Press F9 (or your preferred Shortcut Key) to switch between currently selected language and last selected language.
Note: Typing is case-sensitive. Typing small "a" and capital "A" will yield different letters. Make sure "Caps Lock" key is not ON.

Symbol

The English symbols [ ] { } ( ) < > - + / = ; , ' " ? ! % \ & $ @ # * ^ are transliterated into the same symbols.
Symbols ~ : | . _ have special meaning. You can type this way
~ = ~-/~~

: = :-

| = |-

. = .- (Dot Dash)

_ = _- (Underscore Dash)

These symbols will type Santali characters first but if "-" will be followed, it will remove previously typed Santali character and then type the symbol.

Rule

  1. Consonant is followed by vowel character.
    Example
    ᱠᱚ ᱠᱟ ᱠᱤ ᱠᱩ ᱠᱮ ᱠᱳ
    ko ka ki ku ke kO
  2. Consonant can be followed by consonant character.
    Example
    ᱱᱚᱢᱚᱥᱠᱟᱨ
    nomoskar
