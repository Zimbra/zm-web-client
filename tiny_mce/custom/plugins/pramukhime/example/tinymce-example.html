How to use?

  1. Download TinyMCE 4.x and unzip on hard drive (TinyMCE_ROOT)
  2. Unzip PramukhIME TinyMCE Plugin into "TinyMCE_ROOT\js\tinymce\plugins\" folder.
  3. If you are using TinyMCE Dev Package then place this file example.html into "TinyMCE_ROOT\tests\manual" folder or change the path of "tinymce.js" in this file.
  4. If you are using TinyMCE Production package then place this file example.html into "TinyMCE_ROOT\" folder or change the path of "tinymce.min.js" in this file.
  5. Place this file example.html into "TinyMCE_ROOT\tests\manual" folder or change the path of "tinymce.js/tinymce.min.js" in this file.
  6. Open this file using real webserver like IIS/Apache and you should be able to view PramukhIME drop down list and buttons.
  7. Open this file in text editor and change the options given as per comments within html code to customize PramukhIME plugin.
