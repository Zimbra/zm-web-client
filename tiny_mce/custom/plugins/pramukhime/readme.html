About PramukhIME TinyMCE Plugin

What is PramukhIME TinyMCE Plugin?

PramukhIME TinyMCE Plugin is a plugin for TinyMCE HTML editor using which any Web developer can enable the visitors to type in Indian language without downloading/installing any special software.

About Developer

Gettting inspiration and blessings from H.D.H. Pramukh Swami Maharaj, Vishal Monpara has dedicated more than 3775 hours from January 2005 till February 2018 to create utilities for typing in Indian languages.

Supported TinyMCE version

TinyMCE 4.x is supported.

Version

This plugin as a whole is marked as version 4.0.2. This plugin uses PramukhIME Javascript Library 4.0.1.

Plugin Installation

  • Unzip the file on your local hard disk.
  • Upload/copy "pramukhime" folder to TinyMCE_ROOT/plugins/ so the folder becomes TinyMCE_ROOT/plugins/pramukhime/ (PRAMUKHIME_ROOT)
  • In the init function for TinyMCE in the Webpage, add "pramukhime" in a plugin list and add "pramukhime", "pramukhimesettings", "pramukhimeresetsettings", "pramukhimetogglelanguage" and "pramukhimehelp" buttons for the language dropdown list, keyboard settings, reset keyboard settings, toggle between the current and last selected language and help buttons respectively

Plugin Customization

If you want to customize a list of languages to be displayed in dropdown list, change toggle key and for advanced options, add "pramukhime_options" parameter in tinymce.init() function. Follow the steps given in "example/example.html file
